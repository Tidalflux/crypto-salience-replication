# Replication Notes

## CoinGecko Free API Disclaimer

本复刻研究现阶段切换为 CoinGecko 免费公共 API 的最近 365 天方案，样本窗口为 2025 年 6 月 15 日至 2026 年 6 月 14 日，优先覆盖市值排名前 150 的币种。由于免费 API 的历史区间限制及数据获取难度，本研究未能完整包含原论文中的已下架币种（delisted coins），且数据来源与交易所聚合方式存在差异。因此，本复刻主要用于验证 STR 和 STV 指标构建方法的可行性，以及核心实证结果的方向一致性，而非追求与原论文完全一致的数值结果。所有局限性已在报告中说明。

运行烟测时，CoinGecko 公共 API 返回 `error_code: 10012`，提示免费用户的历史区间查询可能限制在过去 365 天内。因此，当前数据管线以最近一年窗口为准，适合先完成 STR 和 STV 的构建与预测检验。
